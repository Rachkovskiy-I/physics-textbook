ТОМ 4. ОПТИКА. ГЛАВА 4: ОПЫТ МАЙКЕЛЬСОНА-МОРЛИ

Поиск эфирного ветра: доказательство постоянства скорости света
Логика интерферометра:
t_горизонт = 2L / (c * (1 - v²/c²))
t_вертикал = 2L / (c * КОРЕНЬ(1 - v²/c²))

Отрисовка экрана (Детектора):
Δt = t_горизонт - t_вертикал
Яркость = Cos(Δt * Частота_Света)
// В реальности Δt всегда равна 0,
// так как гипотеза эфира (v) ложна!
Скорость ветра: 0.00 c
