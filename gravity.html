Интерактивная симуляция закона тяготения Ньютона (Метод Эйлера)
Параметры такта (dt):
ЦИКЛ_АНИМАЦИИ:
  dx = Солнце.x - Планета.x
  dy = Солнце.y - Планета.y
  r = КОРЕНЬ(dx² + dy²)
  Сила = G * (M * m) / r²
  Ускорение = Сила / m
  Планета.vx += Ускорение * (dx / r) * dt
  Планета.vy += Ускорение * (dy / r) * dt
  Планета.x += Планета.vx * dt
  Планета.y += Планета.vy * dt
